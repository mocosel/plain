TOPIC
    Node.

TYPE
    <Value>.

CATEGORY
    <Naked>.

DEPENDENCIES
    <Statement>,
    <Keyword>,
    <Value>.

USAGE
    1) [[<Statement>]],
    2) <Keyword>.

REMARKS
    Nodes shall be passed by arguments.

    1) <Statement> must be <Declaration> or <Definition>. <Statement> shall be
    expanded to the value it returns.
    2) Translation to [[<Keyword>]] can be supressed. [[<Keyword>]] shall be
    expanded to any <Value> it returns but <Function>.
