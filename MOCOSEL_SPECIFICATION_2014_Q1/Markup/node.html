TOPIC
    Node.

TYPE
    <Value>.

CATEGORY
    <Naked>.

DEPENDENCIES
    <Statement>,
    <Keyword>,
    <Value>.

USAGE
    1) [[<Statement>]],
    2) <Keyword>.

REMARKS
    Nodes shall be passed by arguments.

    1) <Statement> must be <Declaration> or <Definition>. <Statement> shall be
    expanded to any <Value> it returns. If <Statement> expands to <Function>,
    value of the function shall be returned.
    2) <Keyword> must be <Declaration> or <Definition>. <Keyword> shall be
    expanded to any <Value> it returns.
